Macro direct_slot

direct_slot

Synopsis

// In header: <boost/object/slot_macros.hpp>

direct_slot(NAME, IMPL)

Description

Declare a direct slot named NAME. The slot implementation will be IMPL.

If no parameter IMPL is provided, it will default to _slot_NAME

Possible usage: 

 direct_slot(foo)     //The slot implementation will be _slot_foo
 direct_slot(foo,bar) //The slot implementation is bar

Define BOOST_OBJECT_NO_SHORT_SLOT_MACROS to disable this macro.
