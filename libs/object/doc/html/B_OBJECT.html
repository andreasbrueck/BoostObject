Macro B_OBJECT

B_OBJECT

Synopsis

// In header: <boost/object/object.hpp>

B_OBJECT(CLASSNAME)

Description

This macro defines dependencies for your class that other utilities of Boost.Object ( signal/slots/etc ) need.

Possible use: 

 class myClass1 : public BObject
 {
     B_OBJECT(myClass1)

     // ..
 };

 template < typename TARG1, typename TARG2, typename... TARGREST >
 class myClass2 : public BObject
 {
     B_OBJECT(myClass2<TARG1,TARG2,TARGREST...>)

     // ..
 };

Parameters:

CLASSNAME

The type of your class
