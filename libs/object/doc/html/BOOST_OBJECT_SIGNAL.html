Macro BOOST_OBJECT_SIGNAL

BOOST_OBJECT_SIGNAL

Synopsis

// In header: <boost/object/signal_macros.hpp>

BOOST_OBJECT_SIGNAL(NAME, PROTOTYPE)

Description

Declares a signal named NAME which will evaluate as a function of type PROTOTYPE.

Possible usage: 

 BOOST_OBJECT_SIGNAL(foo,void(int))

Parameters:

NAME

The name of your signal

PROTOTYPE

The function prototype this signal shall have
