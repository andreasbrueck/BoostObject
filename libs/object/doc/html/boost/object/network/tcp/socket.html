Description

The base socket connection class for TCP.

socket public construct/copy/destruct

  1. explicit socket(unspecified p);

    Constructor for internal extensions of this class.

socket protected member functions

  1. auto_slot void start();

    Starts reading the socket. Call this function after you connected all your signals.

    This function has no effect if the socket is not connected. 

  2. auto_slot void write(const utils::bytearray & data);

    Writes data to the socket.

    This function has no effect if the socket is not connected.

    Using this function will start reading on this socket, if its not reading yet.

    Parameters:

    data

    The data to be written 

  3. auto_slot void disconnect();

    Disconnects the socket connection.

    This function has no effect if the socket is not connected. 

  4. auto_slot bool noDelay();

    Gets the current setting of the TCP_NODELAY socket option. 

  5. auto_slot void setNoDelay(bool value);

    Sets the TCP_NODELAY socket option.

    Parameters:

    value

    True to enable, false to disable TCP_NODELAY 

  6. signal void received(const utils::bytearray & data);

    This signal is emitted when new data has been read by the socket.

    Parameters:

    data

    The data that was read. 

  7. signal void written(const utils::bytearray & data);

    This signal is emitted after data was written to the socket.

    This data might not be the same as you gave to the write function. The data is internally merged with existing data that has not been sent yet.

    Parameters:

    data

    The data that was written. 

  8. signal void disconnected(const boost::system::error_code & ec);

    This signal is emitted once when a socket is disconnected.
