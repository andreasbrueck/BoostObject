Type PickBehavior

boost::object::threadpool::PickBehavior

Synopsis

// In header: <boost/object/threadpool.hpp>


enum PickBehavior { PickIncrement, PickLeastUsed };

Description

Defines the behavior with which the next thread will be picked.

PickIncrement

Increment an internal counter that restarts at 0 if bigger than the amount of threads

PickLeastUsed

Pick the thread that has the least amount of bobjects assigned.
