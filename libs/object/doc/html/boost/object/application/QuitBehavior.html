Type QuitBehavior

boost::object::application::QuitBehavior

Synopsis

// In header: <boost/object/application.hpp>


enum QuitBehavior { QuitDefault, QuitManual };

Description

Defines when exec() will return

QuitManual

exec() will return if:

  • quit() is called

  • SIGINT or SIGTERM is emitted exec() will only return if you manually call quit()
