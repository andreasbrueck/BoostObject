Description

Emits when native signals such as SIGTERM or SIGINT are delived by your OS.

This class is a wrapper around Boost.Asio's signal_set. Please also consult its documentation on how it handles native signals.

You can have multiple instances of systemsignals to wait for the same signal number(s).
