Macro auto_slot

auto_slot

Synopsis

// In header: <boost/object/slot_macros.hpp>

auto_slot(NAME, IMPL)

Description

Declare an automatic slot named NAME. The slot implementation will be IMPL.

If no parameter IMPL is provided, it will default to _slot_NAME

Possible usage: 

 auto_slot(foo)     //The slot implementation will be _slot_foo
 auto_slot(foo,bar) //The slot implementation is bar

Define BOOST_OBJECT_NO_SHORT_SLOT_MACROS to disable this macro.
