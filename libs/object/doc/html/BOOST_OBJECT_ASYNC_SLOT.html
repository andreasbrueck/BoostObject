Macro BOOST_OBJECT_ASYNC_SLOT

BOOST_OBJECT_ASYNC_SLOT

Synopsis

// In header: <boost/object/slot_macros.hpp>

BOOST_OBJECT_ASYNC_SLOT(NAME, IMPL)

Description

Declare an asynchronous slot named NAME. The slot implementation will be IMPL.

If no parameter IMPL is provided, it will default to _slot_NAME

Possible usage: 

 BOOST_OBJECT_ASYNC_SLOT(foo)     // The slot implementation will be _slot_foo
 BOOST_OBJECT_ASYNC_SLOT(foo,bar) // The slot implementation is bar
