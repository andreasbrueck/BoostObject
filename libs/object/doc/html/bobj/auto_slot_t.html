Description

Template Parameters

  1. typename... FUNCTION_SIGNATURE

    The signature of the slot. Can be a function signature or a list of parameters, the first being the return type:

    • auto_slot_t<void(int)>

    • auto_slot_t<void> Evaluates to auto_slot_t<void()>

    • auto_slot_t<void,int> Evaluates to auto_slot_t<void(int)>

    • auto_slot_t<> Evaluates to auto_slot_t<void()>

auto_slot_t public member functions

  1. RET operator()(ARGS... args);
    Calls the slot implementation in a manner depending from which thread you call this function from.
